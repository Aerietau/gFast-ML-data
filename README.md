# gFast-ML-data
Data used for gFast ML project

Random data was swiped from various sources and ingested into ML models.
One model was to find the ideal customer using distance to fiber with some population density and demographic factors.
One model was to find ideal wireless (5G and Wi-Fi only) customer, using same weights as above but competitively.
Each directory contains the model data ingested as a source for the outputs.
